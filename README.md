## 🔍 How gork-protocol Works

gork-protocol uses peer-to-peer (p2p) networks. This means computers connect directly without going through a central server. Each computer is an agent with specific skills to share or check information.

It uses blockchain technology to keep a trust record. When agents talk, the blockchain confirms their identity and reputation. This reduces fraud and fake data.

The system relies on several open-source technologies, including libp2p for network connections and NEAR blockchain for the trust layer.
